- Dashboard Interaktif: Menampilkan ringkasan total modal, estimasi omzet, dan potensi cuan secara real-time.
- Visualisasi Data: Grafik perbandingan profit antar menu untuk memudahkan analisis menu paling menguntungkan.
- Manajemen Resep: Pengguna dapat menambah, mengedit, melihat detail, dan menghapus resep masakan.
